# LGNB - 聊天灵感提取插件

> 记录群聊消息，自动提取灵感，支持自然语言查询与总结

## 简介

LGNB 是一个 AstrBot 插件，专注于从群聊中提取有价值的灵感、想法和讨论。它会持续记录白名单群聊的消息，定期通过 LLM 自动归类灵感，支持每日定时总结推送、按需时间段总结、AI 观点分析、关键词搜索和多种格式数据导出。

## 功能

- **消息持久化** — 所有消息记录在 SQLite 数据库，支持 message_id 去重
- **FTS5 全文搜索** — 高效搜索聊天记录，支持多关键词
- **自动灵感归类** — 达到阈值后自动调用 LLM 提取灵感
- **每日定时总结** — 可配置时间点，自动生成并可选推送至群聊
- **自然语言交互** — @bot 即可用自然语言查询灵感、生成总结
- **AI 看法** — 让 AI 对某段聊天记录发表观点和评价
- **关键词搜索** — 搜索包含特定话题的聊天记录，LLM 整理摘要
- **时间段总结** — 支持指定日期范围进行长期回顾
- **白名单制** — 仅对配置的群聊/会话生效
- **Debug 模式** — 开启后返回 LLM 理解过程和工具调用链
- **数据导出** — 管理员可将全部数据或指定时间段数据导出为 JSON / Markdown / CSV
- **数据清理** — 管理员可删除指定范围消息；支持自动过期清理
- **用户隔离** — 可配置普通用户仅查询与自己相关的灵感
- **回复分段** — 超长回复自动拆分，兼容平台限制
- **即时反馈** — 先提示"正在思考"，再返回结果

## 配置

在 AstrBot WebUI 插件管理页面进行配置：

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| `whitelist_groups` | 白名单群聊 UMO（用 `/sid` 获取） | `[]` |
| `sub_admins` | 副管理员用户 ID | `[]` |
| `message_threshold` | 累积 N 条消息后自动归类（0=关闭） | `50` |
| `daily_summary_hour` | 每日总结触发小时 | `20` |
| `daily_summary_minute` | 每日总结触发分钟 | `0` |
| `push_daily_summary` | 每日总结是否推送群聊 | `true` |
| `debug_mode` | 是否显示 LLM 调试信息 | `false` |
| `llm_provider` | LLM 提供商（留空使用默认） | `""` |
| `max_query_messages` | 单次检索最大消息数 | `200` |
| `max_reply_length` | 回复最大字符数（0=不限，超长自动分段） | `0` |
| `data_retention_days` | 消息保留天数（0=永久，每日凌晨自动清理） | `0` |
| `query_scope` | 普通用户查询范围：`all` / `self_only` | `all` |

## 使用方式

在群聊中 @bot 或私聊，用自然语言交互：

- **查询灵感**：`最近有什么灵感` / `看看这周的灵感`
- **生成总结**：`总结一下上周的聊天` / `总结 5 月 1 日到 5 月 10 日`
- **查看统计**：`看看数据` / `数据状态`
- **AI 看法**：`你怎么看上周的聊天` / `对 5 月的聊天有什么评价`
- **关键词搜索**：`帮我想想提到 Python 的时候聊了什么` / `搜索「部署」和「Docker」`（多关键词逗号分隔）
- **手动归类**（管理员）：`归类` / `手动归类一下`
- **导出数据**（管理员）：`导出全部数据 markdown格式` / `导出 5 月数据为 csv`
- **删除数据**（管理员）：`删除 4 月 1 日到 4 月 30 日的数据`

## 权限说明

- **管理员**：AstrBot 全局管理员 + 插件配置的副管理员
- **管理员专属**：手动归类、数据导出、数据删除
- **普通用户**：查询灵感、生成总结、AI 看法、搜索记录、查看统计

## 数据存储与导出

所有数据存储在插件目录下 `data/lgnb_data.db`（SQLite），导出文件在 `data/exports/` 目录。

- **JSON 导出**：完整结构化数据
- **Markdown 导出**：可读格式，适合存档阅读
- **CSV 导出**：仅消息，可用 Excel 打开
- **按时间范围导出**：导出指定日期段的数据

## 安装

将插件目录放入 AstrBot 的 `addons/plugins/` 下，在 WebUI 中加载并配置即可。
